# PjDroid

An Android Library for PJSIP PJSUA2 API

## How to use PjDroid

Add this to your app's build.gradle file in dependencies

`implementation 'com.mahmudur85.lib:PjDroid:0.0.1'`

## Feature

Supported architectures

- amreabi-v7a
- arm64-v8a

Support Transport

- UDP
- TCP
- TLS

Supports SRTP
